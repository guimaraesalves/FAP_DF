A redundância de dados não controlada, faz com que
ocorra:
•

dos mesmos dados aparecem em diversos locais. Os da
dos que são
inconsistentes também são chamados de
desprovidos de integridade de
dados
. INTEGRIDADE DE DADOS é definida como a condição e
m que
todos os dados do banco são consistentes com os eve
ntos e condições
reais.
•
Anomalia
